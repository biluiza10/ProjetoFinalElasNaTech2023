Aprendizado

HTML-CSS-JAVA SCRIPT

HTML é o responsável pelo conteúdo das páginas e diferente do que pensava não é uma linguagem de programação e sim uma linguagem de marcação de texto.
Por sua vez e não menos importante o CSS linguagem de estilo, nele definimos (cor) objetos, tamanho, fonte, posicionamento no layout, animação relacionado ao estilo da página.
O JAVA SCRIPT é o responsável por dar vida aos estilos das páginas. Html,Css e Java Script são linguagens muito utilizadas pelos desenvolvedores Front End.

Git-GitHub-Bootstrap-Pyton

GitHub uma Plataforma que permite a hospedagem dos nossos projetos e através do Git que é um software nele conseguimos fazer alterações no nosso projeto e essas alterações ficam salvas sem a necessidade que ficarmos criando pastas referente a cada modificações e também nos permite compartilhar os nossos projetos para que outras pessoas possam trabalhar no nosso código.
Python é uma linguagem de programação muito utilizada em diversos sites, assim como Java tem a sua magia, serve par deixar o HTML mais dinâmico, muito utilizado pelos desenvolvedores Back End.

Sass - Lógica de Programação-JQuery

Sass deixa o CSS com superpoderes é fundamental para projetos grandes, permite fazer modificações de maneira mais rápida, compilar o código nos mostra no terminal prompt de comando onde há erro e facilita a correção.
JQuery um plugin tipo uma biblioteca Java Script que compila os códigos para uma melhor funcionalidade.
Lógica de Programação onde programamos os algoritimos é ela quem organiza de maneira coerente a sequência de ações que um algoritmo precisa executar, ou seja a sequencia de informações para realizar um trabalho.
